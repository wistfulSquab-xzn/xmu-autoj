# XMUOJ 自动答题

## 使用方法

### 双击 `启动.bat` 即可

全程问答式操作，只需回答：

1. **首次使用**：输入学号、密码、比赛密码（仅一次，之后自动记住）
2. **输入比赛 ID**：如 `362`
3. **选择题型**：
   - `1` 全部题目
   - `2` 指定区间（如 `27-40`）
   - `3` 指定题目（如 `JD027,JD029,7318`）
   - `4` 只做前 N 题
4. **确认开始**：回车即跑

---

## 命令行调用（可选）

```powershell
python start.py                              # 交互模式
python start.py -c 361 -p JD027,JD029,7318   # 指定题目
python start.py -c 361 -r 27-40              # 区间
python start.py -c 362 -n 2                  # 前N题
```
